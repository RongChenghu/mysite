# 🤖 AI 工具中心  
> 这里，是我将创意与技术结合的AI实验场。  
> 从绘本到写作，从Prompt优化到产品方案生成，每一个工具都体现了我对「AI + 产品」的理解与实践。

---

## 🧭 模块概览

| 模块 | 功能定位 |
| ---- | ---------- |
| [绘本工坊](./picturebook.md) | 用AI生成孩子的专属旅行绘本 |
| [Prompt优化引擎](./prompts.md) | 让AI更懂你的指令 |
| [AI方案工厂](./solution-factory.md) | 三分钟生成完整产品方案 |
| [RCH Insight](./assistant.md) | 专属AI助理，懂你思考的职业伙伴 |
| [智能写作伴侣](./writing-companion.md) | 分析并生成你的博客内容 |

## 💡 设计理念

> “AI 工具模块不是功能合集，而是品牌的延伸。”

每个工具代表我在不同领域的探索方向：

| 方向 | 工具 | 展示价值 |
| ---- | ---- | -------- |
| 创意与情感 | 绘本工坊 | 展示AI在亲子教育与创意内容的温度感 |
| 系统化思维 | Prompt优化引擎 | 展现Prompt工程与AI理解力的专业度 |
| 产品化能力 | AI方案工厂 | 显示对行业系统与架构思维的掌握 |
| 智能交互 | RCH Insight | 展示如何把AI融入个人知识体系 |
| 内容生态 | 智能写作伴侣 | 体现持续创作与AI共创的长期力 |
