## 🌍 工具简介

## 🧩 功能亮点
- 🧱 产品结构自动生成（功能、数据、交互）  
- 🎯 商业价值提炼与痛点匹配  
- 📋 一键导出PRD / Markdown  
- 🧮 竞品分析模板与对比  
- 📊 自动生成版本迭代路线图

## ⚙️ 技术原理
- 行业知识图谱（能源 / 支付 / 教育 / 供应链）  
- 结构思维映射：自然语言 → 模块树  
- 自动化PRD骨架生成引擎  
- 版本化逻辑：生成V1.0、V2.0发展路线  

---

## 🚀 使用方式
1. 输入行业、目标用户、功能方向  
2. 选择生成类型（方案 / PRD / 竞品分析）  
3. 点击生成 → 自动输出蓝图  
4. 导出或保存
